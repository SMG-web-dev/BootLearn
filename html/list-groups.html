Grupos de Lista

Los grupos de lista son componentes flexibles que permiten mostrar series de contenido, desde simples listas hasta elementos complejos con badges y contenido personalizado.

Lista Básica

Estructura simple

  • Elemento uno
  • Elemento dos
  • Elemento tres
  • Elemento cuatro
<ul class="list-group">
  <li class="list-group-item">Elemento uno</li>
  <li class="list-group-item">Elemento dos</li>
  <li class="list-group-item">Elemento tres</li>
  <li class="list-group-item">Elemento cuatro</li>
</ul>

Lista Activa

Estados y selección

  • Elemento activo
  • Elemento normal
  • Elemento deshabilitado
<ul class="list-group">
  <li class="list-group-item active">Elemento activo</li>
  <li class="list-group-item">Elemento normal</li>
  <li class="list-group-item disabled">Elemento deshabilitado</li>
</ul>
