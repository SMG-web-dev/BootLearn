Scrollspy

Scrollspy actualiza automáticamente la navegación o la lista de elementos basándose en la posición de desplazamiento.

Scrollspy Básico

Con navegación

  • Primera
  • Segunda

Primera sección

Este es un ejemplo de cómo funciona el scrollspy. Al desplazarse, la navegación se actualiza.

Segunda sección

Contenido adicional para mostrar el funcionamiento del scrollspy al desplazarse. 

<nav id="navbar-example2" class="navbar bg-light px-3">
  <ul class="nav nav-pills">
    <li class="nav-item">
      <a class="nav-link" href="#scrollspyHeading1">Primera</a>
    </li>
  </ul>
</nav>
<div data-bs-spy="scroll" data-bs-target="#navbar-example2">
  <h4 id="scrollspyHeading1">Primera sección</h4>
  <p>Contenido...</p>
</div>
