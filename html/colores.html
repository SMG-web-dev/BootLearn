Colores en Bootstrap 5

Bootstrap 5 proporciona una serie de clases de colores que puedes utilizar para dar estilo a tu contenido. Estas clases se dividen en dos categorías: colores de texto y colores de fondo.

Colores de Texto

Este texto representa alguna información.

Este texto representa peligro.

Texto secundario.

Texto gris oscuro.

Texto del cuerpo (predeterminado).

Texto gris claro.

  <p class="text-info">Este texto representa alguna información.</p>
  <p class="text-warning">Este texto representa una advertencia.</p>
  <p class="text-danger">Este texto representa peligro.</p>

  <p class="text-dark">Texto gris oscuro.</p>

  <p class="text-light">Texto gris claro.</p>
