Métodos de Instalación

Bootstrap 5 ofrece múltiples formas de instalación para adaptarse a diferentes flujos de trabajo. Elige el método que mejor se ajuste a las necesidades de tu proyecto.

CDN

La forma más rápida de empezar

Incluye Bootstrap directamente en tu HTML usando el CDN. No necesitas descargar ni instalar nada. 

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
El CDN proporciona mejor velocidad de carga y aprovecha el caché del navegador.

npm

Instalación mediante gestor de paquetes

Ideal para proyectos que utilizan npm como gestor de paquetes y bundlers modernos. 

npm install bootstrap@5.3.0

Importa en tu archivo JavaScript:

import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap/dist/js/bootstrap.bundle.min.js';
Perfecto para proyectos que usan bundlers como Webpack o Vite.

Consejo Pro

Para un mejor rendimiento en producción, asegúrate de minificar los archivos CSS y JavaScript. La versión CDN ya viene minificada por defecto.
