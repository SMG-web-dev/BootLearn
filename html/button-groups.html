Grupos de Botones en Bootstrap 5

Bootstrap 5 permite agrupar una serie de botones juntos (en una sola línea) en un grupo de botones. Utiliza un elemento <div> con la clase .btn-group para crear un grupo de botones.

Ejemplo de Grupo de Botones


<div class="btn-group">
  <button type="button" class="btn btn-primary">Apple</button>
  <button type="button" class="btn btn-primary">Samsung</button>
  <button type="button" class="btn btn-primary">Sony</button>
</div>

Grupos de Botones Anidados

Tablet Smartphone

  <button type="button" class="btn btn-primary">Samsung</button>
  <div class="btn-group">
