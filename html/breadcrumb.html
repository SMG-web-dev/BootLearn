Breadcrumb

El breadcrumb indica la ubicación actual del usuario dentro de una jerarquía de navegación y permite regresar a niveles anteriores.

Breadcrumb Básico

Estructura simple

  1. Inicio
  2. Biblioteca
  3. Datos 
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Inicio</a></li>
    <li class="breadcrumb-item"><a href="#">Biblioteca</a></li>
    <li class="breadcrumb-item active">Datos</li>
  </ol>
</nav>

Divisores Personalizados

Separadores alternativos

  1. Inicio
  2. Categorías
  3. Productos 
<nav style="--bs-breadcrumb-divider: '>';" aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Inicio</a></li>
    <li class="breadcrumb-item"><a href="#">Categorías</a></li>
    <li class="breadcrumb-item active">Productos</li>
  </ol>
</nav>
