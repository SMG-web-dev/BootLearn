Tooltips en Bootstrap 5

El componente Tooltip es un pequeño cuadro emergente que aparece cuando el usuario mueve el puntero del mouse sobre un elemento. Se utiliza para proporcionar información adicional.

Tooltip Básico


<button type="button" class="btn btn-primary" data-bs-toggle="tooltip" title="¡Hooray!">Hover over me!</button>

Inicializar Tooltips


<script>
var tooltipTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="tooltip"]'));
var tooltipList = tooltipTriggerList.map(function (tooltipTriggerEl) {
  return new bootstrap.Tooltip(tooltipTriggerEl);
});
</script>
