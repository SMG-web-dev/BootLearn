Jumbotron

Aunque el componente Jumbotron fue eliminado en Bootstrap 5, puedes crear elementos similares utilizando las utilidades de Bootstrap para destacar contenido importante.

Jumbotron Básico

Estructura simple

Ejemplo de Jumbotron

Utiliza clases de utilidad para recrear el antiguo componente jumbotron y mostrar contenido destacado. 

<div class="p-5 mb-4 bg-light rounded-3">
  <h1 class="display-5 fw-bold">Ejemplo de Jumbotron</h1>
  <p class="fs-4">Utiliza clases de utilidad para recrear el antiguo componente jumbotron y mostrar contenido destacado.</p>
  <button class="btn btn-primary btn-lg" type="button">Ejemplo de botón</button>
</div>

Jumbotron con Color

Ejemplo utilizando color de fondo y texto en blanco para mayor contraste. 

<div class="p-5 mb-4 bg-primary bg-opacity-75 text-white rounded-3">
  <h1 class="display-5 fw-bold">Jumbotron con Color</h1>
  <p class="fs-4">Ejemplo utilizando color de fondo y texto en blanco para mayor contraste.</p>
  <button class="btn btn-light btn-lg" type="button">Llamada a la acción</button>
</div>
