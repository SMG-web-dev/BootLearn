Utilidades en Bootstrap 5

Bootstrap ofrece una serie de clases de utilidad para personalizar el diseño, facilitando la aplicación de estilos sin necesidad de CSS adicional.

Estas clases permiten ajustar márgenes, rellenos, colores y más, lo que te ayuda a crear un diseño más limpio y organizado.
